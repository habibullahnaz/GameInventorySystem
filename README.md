Game Inventory Manager Using Linked Lists

A simple C++ project for managing game inventory items dynamically

## Project Overview

This project implements a Game Inventory Management System using Singly Linked Lists in C++.

It allows the user to:

# Add new items (ID, Name, Quantity)

# Remove items by ID

# Display all inventory items

# Manage the inventory dynamically without using arrays.

The project is mapped with CLO 5.1, demonstrating the correct use of data structures to solve real-world computing problems.
